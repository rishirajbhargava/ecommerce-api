ECOMMERCE API

Welcome to Ecommerce Api

API Description

This is a simple REST API for Ecommerce website/products api. It is built with NodeJs and MongoDB database. It has the following features:

  • Get all products stored in database
  • Get a single product by Id stored in database
  • Add a new product to database
  • Delete a product by Id from database
  • Update a existing product in database
    •

    Tech & Libaries used:

    Nodejs ExpressJs MongoDB Mongoose Postman VS Code Git & Github Nodemon HTML CSS Javascript Google Fonts Font Awesome

    API Endpoints

  • Get all product :
    Route - /api/products
    Method - GET
  • Get Single Product:
    Route - /api/products/{id}
    Method - GET
  • Update a product by Id:
    Route - /api/products/{id}
    Method - PUT
  • Delete a product by Id:
    Route - /api/products/{id}
    Method - DELETE

    • MongoDB Model

    MongoDB is used as the database. The database name is ecommerce and the collection name is products. The model for the product is as follows:

  • product_id :
    Unique Id of the product (manually given)
  • name:
    Name of the product (can be anything)
  • description:
    Descripion of the product.
  • price:
    Price of the product (should be a number)
